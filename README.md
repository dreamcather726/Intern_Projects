# Arduino项目集

这是一个包含多个Arduino和Python项目的综合项目集。

## 📁 项目功能简介

### 🏠 智能家居
- 灯光控制、温湿度监测、窗户控制
- 语音控制、云端连接、图形界面

### 🌱 生物培育
- 光照控制、风扇控制、自动喂食
- 环境参数监控、定时任务

### 🚗 手势小车
- 手势识别控制小车移动
- 串口屏显示状态

### 🌤️ 气象站
- 温湿度、气压等气象数据采集
- 数据显示和记录

### 🏫 广东汇景学校沙盘
- 学校模型控制系统
- 天气播报、环境监测、小车控制

### 🗺️ 世界地图
- 地理信息展示系统
- 音频解说、交互控制

### 💻 硬件代码
- 各种硬件模块基础代码
- MP3播放器、OLED显示、传感器等

## 📋 温州实习项目

### 单文件python
- 简单Python项目，单文件实现

### 多文件python
- 模块化Python项目，多文件结构

### 整理前/整理后
- 项目整理前后的版本对比

---

**说明**：各项目使用Python + Arduino技术栈，包含图形界面和硬件控制功能。

# Arduino项目集

这是一个包含多个Arduino和Python项目的综合项目集，涵盖了智能家居、生物培育、手势控制、气象站等多个领域的应用。

## 📁 项目结构

### 🏠 智能家居 (`智能家居/`)

- **功能**：完整的智能家居控制系统
- **技术栈**：Python + PyQt5 + Arduino + LoRa通信
- **主要特性**：
  - 灯光控制（RGB灯带）
  - 温湿度监测
  - 窗户控制（舵机）
  - 语音控制
  - 云端连接（Blinker平台）
  - 图形化界面

### 🌱 生物培育 (`生物培育/`)

- **功能**：智能生物培育环境监控系统
- **技术栈**：Python + PyQt5 + Arduino + LoRa
- **主要特性**：
  - 光照强度监测与自动控制
  - 风扇控制
  - 自动喂食系统
  - 环境参数实时监控
  - 定时任务管理

### 🚗 手势小车 (`手势小车/`)

- **功能**：基于手势识别的智能小车控制
- **技术栈**：Arduino + 手势传感器（PAJ7620）
- **主要特性**：
  - 手势识别控制
  - 串口屏显示
  - 多方向移动控制

### 🌤️ 气象站 (`气象站/`)

- **功能**：气象数据采集与显示系统
- **技术栈**：Arduino + 传感器
- **主要特性**：
  - 温湿度监测
  - 气压测量
  - 数据记录与分析

### 🏫 广东汇景学校沙盘 (`广东汇景学校沙盘/`)

- **功能**：学校沙盘模型控制系统
- **技术栈**：Arduino + Python
- **主要特性**：
  - 天气播报
  - 环境监测
  - 时间播报
  - 小车控制
  - 语音交互

### 🗺️ 世界地图 (`世界地图/`)

- **功能**：世界地图展示与交互系统
- **技术栈**：串口屏 + MP3音频
- **主要特性**：
  - 地理信息展示
  - 音频解说
  - 交互式控制

### 💻 硬件代码 (`硬件代码/`)

- **功能**：各种硬件模块的基础代码
- **包含模块**：
  - MP3播放器
  - OLED显示屏
  - 土壤数据监测
  - 手势识别
  - 温湿度传感器
  - 编码电机
  - 超声波测距

## 🛠️ 技术特点

### 通信技术

- **LoRa无线通信**：长距离、低功耗的无线数据传输
- **串口通信**：设备间的可靠数据交换
- **云端连接**：通过Blinker平台实现远程控制

### 软件开发

- **Python GUI**：使用PyQt5开发图形化界面
- **多线程处理**：并发处理传感器数据和控制指令
- **模块化设计**：代码结构清晰，易于维护和扩展

### 硬件集成

- **Arduino控制**：多种传感器和执行器的集成控制
- **传感器网络**：多节点数据采集和协同工作
- **实时监控**：数据可视化与状态显示

## 📋 快速开始

### 环境要求

- Python 3.7+
- Arduino IDE
- 相关Python库：PyQt5, pyserial, blinker等

### 运行步骤

1. **安装依赖**：

   ```bash
   pip install PyQt5 pyserial blinker
   ```
2. **配置硬件**：

   - 连接Arduino开发板
   - 安装必要的传感器模块
   - 配置串口参数
3. **运行项目**：

   ```bash
   cd 项目目录
   python main.py
   ```

## 🔧 项目配置

### 串口配置

每个项目都有独立的串口配置文件，需要根据实际硬件连接修改：

- `VOICE_PORT`：语音模块串口
- `LORA_PORT`：LoRa模块串口
- 波特率设置：9600或115200

### 设备ID配置

- 每个Arduino节点有唯一的设备ID
- 在代码中配置需要监控的设备ID列表

### 图片资源

- 各项目包含UI界面所需的图片资源
- 图片路径在代码中配置
